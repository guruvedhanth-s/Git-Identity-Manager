## [1.0.0] - 2026-01-22

### Added
- Initial release of Git Identity Manager CLI
- Multiple Git profile management
- GitHub OAuth Device Flow authentication
- Automatic SSH key generation and upload
- `--profile` flag support for all git commands
- Profile-based repository cloning
- SSH key management per profile
- GPG signing key support
- Profile switching (`git-id use`)
- Profile listing (`git-id list`)
- Current identity display (`git-id current`)
- SSH connection testing (`git-id test`)
- Profile creation via GitHub sign-in (`git-id add --github`)
- Manual profile creation (`git-id add --manual`)
- Profile deletion (`git-id delete`)
- Automatic SSH config management
- Multi-account GitHub workflow support
